## ğŸ¯ Key Interactions

### Creating a Goal
```
1. Click "+ New Goal" button
2. Modal appears
3. Fill in:
   - Name (required)
   - Description (optional)
   - Daily target in minutes (required)
4. Click "Create"
5. Goal appears in list
```

### Starting a Focus Session
```
1. Click on goal from Dashboard
2. Adjust timer settings (optional)
3. Click Play button
4. Timer counts down
5. Browser notification at completion
6. Automatic break timer starts
7. Session recorded
8. Progress updated
```

### Viewing Progress
```
1. Click "Calendar" in navbar
2. See month view with completed days
3. Click on any date
4. See details for that day:
   - Goals completed
   - Minutes spent
   - Coins earned
```
